## 通用概念：断点规则
### 为什么会需要断点规则？
笔者在逆网站的时候逐渐意识到，范围太大、操作不方便Hook起不到高效率节省头发的作用， 
搞这么复杂就是为了**外科手术式**的**精准Hook**。
